## 완료 작업

### 1. 학습 방법 검토 및 방향 결정

- IL 방식은 YOLO 없이 픽셀 패턴 ↔ 행동 직접 학습 → 탑뷰 렌치 검출 불필요

### 2. 카메라 파이프라인 수정
- **문제**: `mjpeg2rgb` 포맷 크래시 → zombie 프로세스가 카메라 장치 점유
- **해결**: `kill -9`로 좀비 제거, `yuyv` 포맷으로 변경
- **신규 파일**: `cameras_v2.launch.py` (pair2 제거, yuyv 포맷)
- **카메라 뷰어**: `camera_viewer_v2.py` + `camera_viewer.sh` (tkinter 기반, cv2 GUI 우회)

### 3. 데이터 수집 (200 에피소드)

- 렌치 위치/각도/손 방향 다양하게 수집
- 전달 범위: cam_top_raw(탑뷰 전체 시야) 안에서 완료
- 렌치 파지: cam_top_bird(600×400mm 매트) 안에서 수행

### 4. 데이터셋 변환 스크립트
- **신규 파일**: `convert_to_lerobot_v3.py`
- 기존 v2 대비 변경: 출력 경로만 wrench_handover_v3으로 수정
- 200 에피소드 전체 → LeRobot 포맷(Parquet + MP4) 변환 예정

### 5. 학습 파이프라인 스크립트
- **신규 파일**: `train_pipeline_v3.sh`
- 변환 → ACT 학습 → Diffusion Policy 학습 순차 실행
- RTX 4070 Laptop(8GB VRAM) 기준 배치 사이즈 32로 조정
